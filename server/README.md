# Push Notifications

## Clone application
`$ git clone https://github.com/nazarvynn/PushNotifications_v2.git`

## Install
### NodeJS >= 10.13.0
### NPM >= 6.4.1
### Postgres >= 7.8
### Knex
`$ sudo npm install -g knex`
### Dependencies
```
$ cd PushNotifications_v2/server
$ npm i
```

## Start Postgres

## DB Migrate
```
$ cd PushNotifications_v2/server
$ knex migrate:latest
```

## Start server
```
$ cd PushNotifications_v2/server
$ npm start
```
